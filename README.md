## Getting started

1. Fork the repo

```
https://github.com/viiralt/weatherbox
```
2. Clone the repo locally. In a terminal window, run: 

```
git clone https://github.com/[your-github-profile-name]/weatherbox.git
```
3. Install dependencies 

```
npm install
```
4. Run the client
```
npm run start
```
